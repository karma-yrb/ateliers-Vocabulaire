📋 Présentation de l'application

Atelier Vocabulaire est une application web autonome (un seul fichier HTML, sans installation, sans connexion internet requise) destinée à l'apprentissage du vocabulaire français en contexte d'atelier sociolinguistique (ASL).

Elle propose deux parcours adaptés à des profils d'apprenants différents, accessibles depuis un ordinateur, une tablette ou un grand écran.

Avantage clé : aucune connexion internet n'est nécessaire. L'application fonctionne entièrement dans le navigateur, sur n'importe quel appareil.

🎯 Les deux parcours

À l'ouverture, l'apprenant·e choisit son parcours. Les intitulés sont volontairement formulés en langage simple, sans jargon pédagogique.

🌱 Je découvre les mots
  • Public visé : personnes peu ou pas scolarisées dans leur pays d'origine (niveau dit « alpha »)
  • Découverte du lien image → mot écrit
  • Pas de saisie au clavier requise
  • Interface très épurée, grands éléments
📖 Je lis et j'écris
  • Public visé : personnes ayant une scolarisation dans leur langue d'origine (public allophone)
  • Apprentissage du vocabulaire avec articles
  • Saisie au clavier, QCM, anagrammes, lecture
  • Statistiques et minuteur de session
Attention : ces deux parcours sont indépendants. L'apprenant·e peut passer de l'un à l'autre à tout moment via le bouton « Changer de parcours ».

🌱 Je découvre les mots

Objectif pédagogique

Associer une image à son mot écrit en français. L'apprenant·e n'a pas besoin de connaître le clavier. Tout se fait avec la souris (ou le doigt sur tablette).

Mode « Construire le mot »

Un emoji s'affiche. En dessous, des tuiles de lettres mélangées sont proposées. L'apprenant·e clique les lettres dans le bon ordre pour former le mot.

  • Cliquer une lettre l'ajoute au mot en cours de construction
  • Cliquer une lettre déjà placée la retire
  • Quand le mot est complet, la validation est automatique
  • En cas d'erreur : les lettres se réinitialisent automatiquement après un bref délai
  • Le bouton Réinitialiser permet de tout effacer à la demande

Mode « Lire seulement »

Le mot s'affiche en gros caractères sous l'emoji. Idéal pour la lecture à voix haute ou la phase de découverte avant l'exercice. Navigation libre avec les boutons Précédent / Mot suivant.

Progression

22 mots répartis en 4 catégories (Corps, Nourriture, Maison, Vie quotidienne). L'ordre est aléatoire à chaque nouvelle session. Un bilan en pourcentage s'affiche à la fin de la série.

Conseil formateur·trice : commencer par le mode « Lire seulement » pour une première prise de contact avec les mots, puis basculer en mode « Construire » pour l'exercice actif.

📖 Je lis et j'écris

Objectif pédagogique

Enrichir le vocabulaire français courant, mémoriser les articles (un, une, du…) et s'entraîner à l'orthographe en contexte visuel.

Les 4 modes d'exercice

Mode Description Compétence travaillée
✏️ Écrire Taper le mot correspondant à l'image dans un champ de saisie. L'article est accepté mais pas obligatoire. Orthographe, mémorisation
🔘 QCM Choisir parmi 4 propositions. Les distracteurs sont tirés au sort dans tout le vocabulaire. Reconnaissance, discrimination lexicale
🔤 Lettres Remettre les lettres mélangées dans le bon ordre en cliquant les tuiles. Conscience de la forme orthographique
👁 Lecture Affichage libre du mot avec l'image. Navigation avec les flèches clavier. Lecture, mémorisation passive

Minuteur de session

Un chronomètre décompteur est disponible dans la barre latérale : 30 min ou 60 min. Il s'arrête automatiquement à zéro. Il peut être arrêté manuellement avec le bouton Stop.

Filtre par catégories

Il est possible de sélectionner une ou plusieurs catégories de mots pour cibler le travail sur un thème particulier (ex. : uniquement « Corps » et « Maison »). Au moins une catégorie doit rester active.

Statistiques

La barre latérale affiche en temps réel :

  • Correct : nombre de bonnes réponses
  • Erreurs : nombre de mauvaises réponses
  • Tentatives : total de validations
  • Réussite : pourcentage de bonnes réponses sur les tentatives

Raccourcis clavier

  • Entrée — Valider la réponse (mode Écrire)
  • → — Mot suivant
  • ← — Mot précédent
Validation souple : en mode Écrire, la réponse est acceptée avec ou sans article. Les accents ne sont pas pénalisés (« elevé » = « élevé »).

💡 Conseils d'utilisation en atelier

Déroulement type d'une séance (60 min)

  1. Découverte (10 min) — Mode « Lire seulement » ou « Lecture » : parcourir les mots d'une catégorie à voix haute avec la formatrice.
  2. Exercice actif (30–40 min) — Mode « Construire » (public alpha) ou « Écrire » / « QCM » (public allophone).
  3. Consolidation (10 min) — Mode « Lettres » pour travailler la forme des mots sans support écrit visible.
  4. Bilan (5 min) — Lire les statistiques, commenter les erreurs, identifier les mots à revoir.

Conseils spécifiques public alpha

  • Verbaliser chaque lettre à voix haute au moment du clic.
  • Ne pas corriger l'absence d'accents dans un premier temps.
  • Privilégier les catégories « Corps » et « Maison » en début de parcours.
  • Laisser l'apprenant·e utiliser la souris de façon autonome.

Conseils spécifiques public allophone

  • Encourager l'écriture de l'article pour travailler le genre grammatical.
  • Utiliser le filtre de catégories pour centrer la séance sur un thème du cours.
  • Le mode QCM est idéal en début de séance pour réactiver le vocabulaire vu la semaine précédente.
  • Le chronomètre peut être affiché pour motiver la régularité sans créer de pression.
En groupe : projeter l'application sur un grand écran et faire répondre les apprenants à tour de rôle. Désactiver le minuteur dans ce cas.

Compatibilité

  • Navigateurs récents : Chrome, Firefox, Edge, Safari (2021 et après)
  • Aucune installation, aucune connexion internet requise
  • Responsive : fonctionne sur tablette et ordinateur
  • Pas d'accessibilité pour les lecteurs d'écran (pas l'usage visé)

Modification du vocabulaire

Les listes de mots se trouvent dans la balise <script> du fichier HTML, dans les tableaux VOCAB_ALPHA et VOCAB_ALLOPHONE. Chaque entrée suit ce format : 

{ emoji: "🍎", word: "une pomme", cat: "Fruits" }

Pour le parcours alpha, le champ word ne contient pas d'article. Pour le parcours allophone, il inclut l'article.

Sauvegarde : l'application ne sauvegarde aucune donnée entre les sessions. Les statistiques sont remises à zéro à chaque ouverture du fichier.
